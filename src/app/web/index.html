Журнал → PostgreSQL · состояние → Redis · метрики → Prometheus / Grafana. Одна и та же цепочка signal → risk → OMS работает в бэктесте, paper- и live-режиме — меняются только источник данных и шлюз исполнения.

Статус компонентов

Market data — реальные исторические данные + IB / mock
Signal engine — SMA crossover
Risk manager — pre-trade checks активны
OMS — учёт заявок/позиций
Execution gateway — paper-симуляция
API /health — проверяю…

Интерактивный бэктест

SMA-crossover — учебная стратегия: демонстрирует движок, а не доходность. Увеличьте qty (напр. до 100) — риск-менеджер (лимит позиции $50 000) начнёт отклонять крупные заявки.
